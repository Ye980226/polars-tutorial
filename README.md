## PR要求
* `fork`之后建立自己的分支，先在自己的分支上修改
* 必须先在`Jupyter notebook`中写，代码运行测试通过
* 每一章notebook下面必须附上写这篇文章的参考文献（采用规定格式）
* 文章需要的图片必须放在`docs/_images/`下，图片采用相对路径
* 数据集请放在`data/`下，命名好名称
* 尽量拓展每一章节的内容，发挥想象多举几个用例
* 项目初期尽量以研读官方文档为主，如果能够把数据分析的项目等改写成Pandas实现，那么后续再加入
